Radiocasty iOS App Beta Test

The app is currently in an early beta version. Follow the steps below to become an iOS Beta Tester.

  1. Install TestFlight from the App Store

    If you haven’t already, install the TestFlight app. Radiocasty app updates come through TestFlight. Make sure notifications are enabled in TestFlight to be prompted when new version is available.

  2. Join Radiocasty App Beta Test Program

    With TestFlight installed, open this link on your mobile device to install the most recent version of the Radiocasty Beta app.

The app requires iOS version 14.5
